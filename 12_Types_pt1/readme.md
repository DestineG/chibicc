## 本章新增支持语法

``` c
char j;
```

**主要改动:**
- 添加 char 类型的识别

## 测试

**测试方法**
``` bash
cd 12_Types_pt1
make clean
make
make test
```

**测试结果**
``` bash
--- 运行结果如下 ---
./build/out
20
10
1
2
3
4
5
253
254
255
0
1
2
3
4
--- 测试完成 ---
```
j 的数据范围是 0~255，之所以后续打印0~4，也是因为 j = j + 1; 右端超过 255，但是赋值表达式只会根据左端的数据类型对右端数据进行截取赋值(左端为 char 就只会截取右端的低 8 位)
